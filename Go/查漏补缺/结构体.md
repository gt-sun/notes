


## 空结构

```go
package main

import (
    "fmt"
    "unsafe"
)

type Q struct{}

func main() {
    var s = "Tim"
    var s3 struct{}
    fmt.Println(unsafe.Sizeof(s))
    fmt.Println(unsafe.Sizeof(s3)) //0

    var a = Q{}
    var b = Q{}
    fmt.Println(&a == &b) //true
}
```

造成这个结果的原因是 Golang 的编译器会把这种空对象都当成`runtime.zerobase`处理。

```
var zerobase uintptr
```
