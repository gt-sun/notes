## `__class__`

```python
class C(object):
    def __init__(self, x):
        self.x = x

c = C(1)

print(c.x)  # 1
print(c.__class__)  # <class '__main__.C'>
#打印类名
print(c.__class__.__name__)  # C

```

## `__repr__`


用来重写对象本身

原生的：
```python
class C(object):
    def __init__(self, x):
        self.x = x

c = C(1)
print(c)  # <__main__.C object at 0x0000011A9AF4C4E0>
```

重写后：
```python
class C(object):
    def __init__(self, x):
        self.x = x

    def __repr__(self):
        return (self.__class__.__name__ + "--> " + ("x={}".format(self.x)))

c = C(1)
print(c)  # C--> x=1
```
