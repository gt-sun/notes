## 事务未提交，导致阻塞

> Lock wait timeout exceeded; try restarting transaction

查看事务表`information_schema.INNODB_TRX`，找到线程 `trx_mysql_thread_id`
再查看`show full processlist`,找到id
`kill id`; 即可。


## mysql 时间与系统时间不一致

查看mysql当前时间
`select now();`

设置mysql时间
`set global time_zone='+8:00';` 
`flush privileges;`

在`my.cnf` 中固定时间设置：

```
[mysqld]
default-time_zone = '+8:00'
```
